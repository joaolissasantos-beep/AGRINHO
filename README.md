
# 🌱 Agrinho 2026 - Agronegócio Equilibrado

Projeto desenvolvido para o Concurso Agrinho 2026 com o tema:
**"Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente"**

## 💡 Sobre o projeto
Este site apresenta a importância do agronegócio equilibrado, destacando como é possível produzir alimentos e, ao mesmo tempo, preservar o meio ambiente.

## 🎯 Objetivo
Mostrar a relação entre produção agrícola e sustentabilidade de forma simples e educativa para estudantes do Ensino Médio.

## 🛠️ Tecnologias usadas
- HTML5
- CSS3
- JavaScript

## 📚 Funcionalidades
- Página informativa sobre agronegócio sustentável
- Explicação de problemas e soluções
- Quiz interativo educativo

## 🌍 Aprendizado
Este projeto ajuda a entender como a tecnologia pode apoiar a educação ambiental e o desenvolvimento sustentável.
